Git/GitHub
Git != GitHub
Git - система контроля версий
GitHub - облачное хранилище

Repository - это хранилище исходных кодов.
fetch, pull и push
fetch - скачивает изменения с удаленного репозитория на локальный;
pull  - скачивает и распаковывает изменения с удаленного репозитория на локальный;
push  - отправляет изменения с локального репозитория на удаленный;

commit - это контрольная точка, которая хранит снимок всех нужных файлов репозитория 
		 за какой-то момент времени.
commit message - это описание внесенных изменений.

branch - это последовательность commit-ов.
master - это главная ветка
merge  - слияние веток
